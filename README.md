# MHelper

AI 驱动的辅助背诵学习助手。

## 功能

- **辅助模式** — 上传 Markdown 笔记，选中不懂的文字和知识点，AI 以通俗语言解释
- **测验模式** — 从笔记中解析问题，默写答案，AI 严格对照参考答案逐句检查背诵

## 使用

1. 打开 `MHelper.html` 或 `agent.html`
2. 在设置中填入 OpenAI 兼容 API（DeepSeek / GPT 等）的地址和密钥
3. 上传 `.md` 格式的学习提纲即可开始学习

## 技术

纯前端单页应用（HTML / CSS / JavaScript），无构建步骤，无后端依赖。
